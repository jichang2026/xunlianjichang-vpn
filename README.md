## ⚠️ 迅连机场为什么不等于瞬连VPN？

### 机场 ≠ VPN。

机场：提供代理节点和订阅服务，需要配合 Clash、Sing-box 等客户端使用。
VPN：提供标准 VPN 隧道连接，通常使用 WireGuard、OpenVPN、IKEv2 等协议。
它们都可以实现类似的网络访问效果，但底层技术、协议和部署方式并不相同。
⭐ 不过话说回来，Soonlink官网叫迅连，但是给你发邮件时却又变成了瞬连，所以让很多人以为它又叫瞬连VPN.
